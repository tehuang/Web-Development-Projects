Middlewares (in between raw request that comes into the server and before it get processed by routes handler (e.g. GET,POST,PUT..))
Server -> Middlewares -> Routes(GET,POST,PUT..)

Advantages of middlewares:

  1. Pre-processing the request e.g. body parser
  2. Logging request e.g. morgan
  3. Authentication
  4. Identify error

E.g. Body Parser - body parsing middleware //Frequently use in form data
